🏡 Airbnb Data Analysis

About the Project

In this project, I explored an Airbnb dataset to understand how different factors like location, room type, and availability affect pricing and bookings.

The main goal was to clean the data and find useful insights that can help in making better business decisions.

---

Dataset Details

- Around 50,000 Airbnb listings
- Includes data like price, location, room type, availability, and reviews

---

Tools Used

- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

---

What I Did

- Cleaned the dataset (handled missing values and duplicates)
- Analyzed how prices vary across different locations
- Studied availability trends
- Explored relationships between different variables using correlation

---

Key Findings

- Entire homes are generally more expensive than shared rooms
- Some locations have consistently higher prices and demand
- Availability of listings has a direct impact on pricing
- Reviews play an important role in booking decisions
